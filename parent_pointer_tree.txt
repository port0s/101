Parent Pointer-Tree
árvore interna ou 
árvore de ponteiros pai 
é uma estrutura de dados de árvore N 
na qual cada nó tem um ponteiro para seu nó pai, 
mas nenhum ponteiro para nós filhos. 

Quando usada para implementar um conjunto de pilhas, 
a estrutura é chamada de pilha de espaguete, 
pilha de cactos ou pilha de sahuaro 
(em homenagem ao sahuaro, uma espécie de cacto). 

Árvores de ponteiros pai também são usadas como estruturas de dados com conjuntos separados.


Pilha de espaguete com uma moldura de pilha "ativa" destacada
A estrutura pode ser considerada 
como um conjunto de listas unidas individualmente 
que compartilham parte de sua estrutura, 
em particular, 
suas caudas. 
De qualquer nó, 
pode-se atravessar para os ancestrais do nó, 
mas não para qualquer outro nó.
